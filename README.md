# TCM AI Agent

一个基于 **本地大模型（Qwen3-32B）+ RAG + 结构化中医知识库 + 推理链** 的中医智能问诊系统。

---

## 🚀 项目亮点

### 1️⃣ 结构化中医知识库

* 构建“症状 → 证型 → 方剂”体系
* 引入：关键症状 / 舌象 / 脉象 / 排除项
* 提升检索精度与可解释性

### 2️⃣ 推理链（核心创新）

不同于传统 RAG：

```text
用户输入
→ 症状抽取
→ 证型打分
→ Top-K排序
→ 检索证据
→ 大模型生成解释
```

实现“类医生推理流程”。

### 3️⃣ 本地大模型（隐私 & 成本友好）

* 基于 vLLM 部署 Qwen3-32B
* OpenAI 兼容接口
* 支持离线运行

### 4️⃣ 医疗安全机制

* 危险症状检测
* 强制提示就医
* 避免误导性输出

## 🧠 系统架构

```text
            ┌──────────────┐
            │ 用户输入症状 │
            └──────┬───────┘
                   ↓
        ┌────────────────────┐
        │ 症状抽取（规则）   │
        └──────┬─────────────┘
               ↓
        ┌────────────────────┐
        │ 证型打分排序       │
        │ (rule-based)       │
        └──────┬─────────────┘
               ↓
        ┌────────────────────┐
        │ RAG 检索           │
        └──────┬─────────────┘
               ↓
        ┌────────────────────┐
        │ Qwen3-32B 生成     │
        └──────┬─────────────┘
               ↓
        ┌────────────────────┐
        │ 结构化输出         │
        └────────────────────┘
```

## ⚙️ 安装与运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置环境变量

```bash
cp .env.example .env
```

---

## 🤖 启动本地模型（vLLM）

```bash
python -m vllm.entrypoints.openai.api_server \
  --model Qwen/Qwen3-32B \
  --served-model-name qwen3-32b \
  --host 0.0.0.0 --port 8000
```

---

## 🖥 启动前端

```bash
streamlit run app.py
```
